# M08L02-DESARROLLO DE PORTAFOLIO DE UN PRODUCTO DIGITAL

## Descripción
Este proyecto es un Producto Digital Funcional consistente en un Blog de Noticias dinámico. El sitio permite la visualización de noticias en tiempo real desde una base de datos y cuenta con un sistema de autenticación de usuarios mediante Google.

## Tecnologías utilizadas
- HTML5 y SASS (Estructura y Estilos)
- Bootstrap 5.3.8 (Maquetación responsiva)
- jQuery 3.7.1 (Manejo de eventos y DOM)
- Firebase Auth (Gestión de usuarios)
- Cloud Firestore (Base de datos NoSQL)

## Instrucciones de uso
1. Clonar el repositorio.
2. Abrir index.html con la extensión Live Server de Visual Studio Code.
3. Para compilar cambios estéticos, usar Live Sass Compiler sobre la carpeta scss.
4. Iniciar sesión con una cuenta de Google para interactuar con el sistema.
